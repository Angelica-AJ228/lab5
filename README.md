# lab5
Repositorio del codigo del laboratorio 5 de Plataformas abiertas

Este codigo es sobre la estructura de datos de arreglos dinamicos. A continuacion se presenta la forma para compilar todo el programa y ejecutarlo. Ademas, se muestra la forma para usar valgrind y clean para eliminar el archivo ejecutable.


Compilacion: una vez descargado todo el programa, asegurate de estar en el folder que contiene el Makefile y usa...

make


Ejecutable: al hacer "make" se crea un archivo ejecutable llamado main. Usa esto para ejecutarlo:

./main

Usar valgrind:

make valgrind

Clean: elimina el archivo ejecutable 

make clean
